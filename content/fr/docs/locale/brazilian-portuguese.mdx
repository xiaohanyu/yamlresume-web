Le portugais brésilien (_português brasileiro_) est une langue romane descendante
du [latin vulgaire](https://en.wikipedia.org/wiki/Vulgar_Latin) et la langue officielle
du Brésil, le plus grand pays d'Amérique du Sud et du monde
[lusophone](https://en.wikipedia.org/wiki/Lusophone). Le portugais et l'espagnol
partagent de nombreuses similitudes en tant que [langues romanes](https://en.wikipedia.org/wiki/Romance_languages)
étroitement apparentées, de sorte que la plupart des fondations communes de
YAMLResume peuvent être réutilisées pour le portugais brésilien.

## Linguistique

### Jeu de caractères

Le portugais brésilien utilise l'[alphabet
latin](https://en.wikipedia.org/wiki/Latin_alphabet) avec 26 lettres (`a-z`).
Contrairement à l'espagnol, le portugais n'a pas la lettre `ñ`. À la place,
l'équivalent nasalisé s'écrit `nh`. Le portugais utilise plusieurs signes
diacritiques pour l'accentuation, la nasalisation et la prononciation :

| Feature                | English Character Set             | Brazilian Portuguese Character Set                                                                                          |
| ---------------------- | --------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| **Alphabet**           | 26 letters (`a-z`)                | 26 letters (`a-z`)                                                                                                          |
| **Diacritics**         | None                              | Includes acute (`á`, `é`, `í`, `ó`, `ú`), circumflex (`â`, `ê`, `ô`), grave (`à`), tilde (`ã`, `õ`), and diaeresis (`ü`)    |
| **Special Characters** | Limited to basic Latin characters | Includes `ç` (c-cedilha), which is not considered a separate letter but a variant of `c`                                     |

Les principales différences entre les systèmes d'écriture de l'anglais et du portugais brésilien :

- **Diacritiques** : le portugais utilise plusieurs signes diacritiques qui indiquent
  l'accent tonique et les changements de prononciation. Par exemple, l'accent aigu (`é`)
  indique les voyelles ouvertes, l'accent circonflexe (`ê`) les voyelles fermées, et
  le tilde (`ã`, `õ`) la nasalisation.
- **Cédille** : le `ç` (c-cédille) est propre au portugais et apparaît uniquement
  avant `a`, `o` et `u` (ex. _caça_, _aço_, _açúcar_).
- **Pas de signes inversés** : contrairement à l'espagnol, le portugais n'utilise pas
  de points d'interrogation (`¿`) ni d'exclamation (`¡`) inversés.

## Expressions idiomatiques

### Date

Les jours de la semaine et les mois ne prennent pas de majuscule en portugais. Par
exemple, « Sep 2021 » en anglais se traduit par « set 2021 » en portugais brésilien.
Les noms de mois sous forme abrégée sont généralement suivis d'un point (ex. `jan.`).

YAMLResume n'utilise que les mois, d'où le tableau de traduction :

| --------------- | -------------------- |
| January (Jan)   | janeiro (jan.)       |
| February (Feb)  | fevereiro (fev.)     |
| March (Mar)     | março (mar.)         |
| April (Apr)     | abril (abr.)         |
| May (May)       | maio (mai.)          |
| June (Jun)      | junho (jun.)         |
| July (Jul)      | julho (jul.)         |
| August (Aug)    | agosto (ago.)        |

| October (Oct)   | outubro (out.)       |

| December (Dec)  | dezembro (dez.)      |

### Capitalisation

Le portugais suit des règles de capitalisation conservatrices similaires à l'espagnol.
Les mois, les jours de la semaine, les langues et les nationalités sont en minuscules.
Contrairement à l'allemand, le portugais ne met pas tous les noms en majuscule.

YAMLResume adopte en interne un ensemble fixe d'options et de termes pour tous les
modèles de CV. Par exemple, les langues dans les modèles de CV YAMLResume adoptent
les [options de maîtrise linguistique de
LinkedIn](https://www.linkedin.com/pulse/language-proficiency-linkedin-duaa-ali/) :

- Elementary Proficiency
- Limited Working Proficiency
- Minimum Professional Proficiency
- Full Professional Proficiency
- Native or Bilingual Proficiency

Celles-ci sont traduites en portugais brésilien comme suit :

- Proficiência Básica
- Proficiência Profissional Limitada
- Proficiência Profissional Mínima
- Proficiência Profissional Fluente
- Nativo ou Bilíngue

Only the first word is capitalized.

## Composition typographique

### Ponctuation

Le portugais brésilien partage plusieurs conventions de ponctuation avec l'anglais,
mais présente quelques différences :

- **Guillemets** : comme en anglais, les points et les virgules sont placés à
  l'intérieur des guillemets en portugais brésilien.
- **Signes inversés** : contrairement à l'espagnol, le portugais n'utilise **pas**
  de points d'interrogation (`¿`) ni d'exclamation (`¡`) inversés.
- **Virgule de série** : le portugais n'utilise pas de virgule avant « e » dans les
  listes (ex. « maçãs, laranjas e bananas »), comme l'espagnol.
- **Formats numériques** : l'anglais utilise un point comme séparateur décimal
  (ex. 3.14), tandis que le portugais utilise une virgule (ex. 3,14). Pour les
  milliers, le portugais utilise un point (ex. 1.000).
