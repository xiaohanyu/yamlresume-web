---
title: レイアウト
---

履歴書の核心はコンテンツですが、レイアウトも重要です。多くの JSON/YAML ベースの履歴書ジェネレーターはコンテンツのみに焦点を当て、レイアウトを変更せずに残し、ユーザーに大きな問題を残しています。

他の履歴書ジェネレーターとは異なり、YAMLResume はすぐに使えるレイアウトソリューションを提供し、ユーザーが履歴書のコンテンツに集中しながら、ピクセルパーフェクトなレイアウトとタイポグラフィでプロフェッショナルな履歴書を生成できるようにします。

以下は `resume.yml` ファイルでのレイアウト設定の例です：

```yml lineNumbers title="resume.yml"
---
content:
  # ...

layout:
  locale:
    language: en
  margins:
    top: 2.5cm
    left: 1.5cm
    right: 1.5cm
    bottom: 2.5cm
  page:
    showPageNumbers: true
  template: moderncv-banking
  typography:
    fontSize: 11pt
```

## ロケール

`locale` オプションにより、履歴書の言語を指定できます。これは日付形式やその他の言語固有の設定に影響します。

```yml lineNumbers
layout:
  locale:
    language: en
```

詳細については [多言語](/ja/docs/content/multi-languages) をご覧ください。

## マージン

`margins` オプションにより、ページマージンをカスタマイズできます。

```yml lineNumbers
layout:
  margins:
    top: 2.5cm
    left: 1.5cm
    right: 1.5cm
    bottom: 2.5cm
```

## ページ

`page` オプションにより、用紙サイズやページ番号の表示など、ページ設定をカスタマイズできます。

```yml lineNumbers
layout:
  page:
    showPageNumbers: true
```

## セクション

`sections` オプションにより、セクションタイトルとその順序をカスタマイズできます。詳細については [セクション文書](./layout/sections) をご覧ください。

```yml lineNumbers
layout:
  sections:
    aliases:
      work: "Professional Experience"
      education: "Education"
      skills: "Skills"
    order:
      - basics
      - work
      - education
```


## テンプレート

`template` オプションにより、様々なテンプレートから選択できます。各テンプレートには独特なデザインとレイアウトがあります。利用可能なテンプレートのリストは [テンプレート文書](./layout/templates) で確認できます。

```yml lineNumbers
layout:
  template: moderncv-banking
```

## タイポグラフィ

`typography` オプションにより、履歴書で使用されるフォントスタイルをカスタマイズできます。

現在、`fontSize` のみサポートされており、LaTeX エンジンは `10pt`、`11pt`、`12pt` のみサポートしています。

```yml lineNumbers
layout:
  typography:
    fontSize: 11pt
```
