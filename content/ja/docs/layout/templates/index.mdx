---
title: テンプレート
---

YAMLResume はいくつかのすぐに使用できるテンプレートを提供しており、`yamlresume templates list` コマンドですべてのテンプレートをリストできます。

```console
$ yamlresume templates list

| `layout.template` | Template Name    | Description                          |
| ----------------- | ---------------- | ------------------------------------ |
| moderncv-banking  | ModernCV Banking | ModernCV template with banking style |
| moderncv-casual   | ModernCV Casual  | ModernCV template with casual style  |
| moderncv-classic  | ModernCV Classic | ModernCV template with classic style |
```

デフォルトでは、YAMLResume は [moderncv-banking](./templates/moderncv-banking) テンプレートを使用します。YAML 履歴書で `layout.template` を設定することでテンプレートを変更できます。

```yml lineNumbers title="resume.yml"
content:
  # ...

layout:
  # ...
  template: moderncv-classic # [!code highlight]
  # ...
```
