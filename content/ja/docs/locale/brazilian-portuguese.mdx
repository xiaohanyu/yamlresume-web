ブラジルポルトガル語（_português brasileiro_）は、[俗ラテン語](https://en.wikipedia.org/wiki/Vulgar_Latin)から派生したロマンス語で、南アメリカ最大かつ[ポルトガル語圏](https://en.wikipedia.org/wiki/Lusophone)最大の国であるブラジルの公用語です。ポルトガル語とスペイン語は、密接に関連する[ロマンス語](https://en.wikipedia.org/wiki/Romance_languages)として多くの類似点を共有しているため、YAMLResumeの共通基盤のほとんどをブラジルポルトガル語に再利用できます。

## 言語学

### 文字セット

ブラジルポルトガル語は、26文字（`a-z`）の[ラテン
アルファベット](https://en.wikipedia.org/wiki/Latin_alphabet)を使用します。
スペイン語とは異なり、ポルトガル語には `ñ` の文字がありません。代わりに、
鼻音化された同等のものは `nh` と表記されます。ポルトガル語は強勢、鼻音化、
発音を示すために複数のダイアクリティカルマークを使用します：

| Feature                | English Character Set             | Brazilian Portuguese Character Set                                                                                          |
| ---------------------- | --------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| **Alphabet**           | 26 letters (`a-z`)                | 26 letters (`a-z`)                                                                                                          |
| **Diacritics**         | None                              | Includes acute (`á`, `é`, `í`, `ó`, `ú`), circumflex (`â`, `ê`, `ô`), grave (`à`), tilde (`ã`, `õ`), and diaeresis (`ü`)    |
| **Special Characters** | Limited to basic Latin characters | Includes `ç` (c-cedilha), which is not considered a separate letter but a variant of `c`                                     |

英語とブラジルポルトガル語の書記体系の主な違い：

- **ダイアクリティカルマーク**：ポルトガル語では、強勢や発音の変化を示すいくつかの
  記号を使用します。例えば、アキュートアクセント（`é`）は開母音を示し、
  サーカムフレックス（`ê`）は閉母音を示し、チルダ（`ã`、`õ`）は鼻音化を示します。
- **セディーユ**：`ç`（c-セディーユ）はポルトガル語特有で、`a`、`o`、`u` の前にのみ
  現れます（例：_caça_、_aço_、_açúcar_）。
- **逆疑問符・逆感嘆符なし**：スペイン語とは異なり、ポルトガル語では逆疑問符（`¿`）
  や逆感嘆符（`¡`）を使用しません。

## 慣用表現

### 日付

ポルトガル語では曜日と月は大文字になりません。例えば、英語の「Sep 2021」は
ブラジルポルトガル語では「set 2021」と翻訳されます。省略形の月名は通常、
ピリオドが続きます（例：`jan.`）。

YAMLResumeは月のみを使用するため、翻訳表：

### 大文字化

ポルトガル語はスペイン語と同様の保守的な大文字化ルールに従います。月、曜日、
言語、国籍は小文字です。ドイツ語とは異なり、ポルトガル語ではすべての名詞を
大文字にしません。

YAMLResumeは内部で、すべての履歴書テンプレートに固定のオプションと用語セットを
採用しています。例えば、YAMLResumeの履歴書テンプレートの言語は
[LinkedInの言語能力オプション](https://www.linkedin.com/pulse/language-proficiency-linkedin-duaa-ali/)を採用しています：

- Elementary Proficiency
- Limited Working Proficiency
- Minimum Professional Proficiency
- Full Professional Proficiency
- Native or Bilingual Proficiency

これらはブラジルポルトガル語に以下のように翻訳されます：

- Proficiência Básica
- Proficiência Profissional Limitada
- Proficiência Profissional Mínima
- Proficiência Profissional Fluente
- Nativo ou Bilíngue

最初の単語のみが大文字になります。

## 組版

### 句読点

ブラジルポルトガル語は英語といくつかの句読点の規則を共有していますが、
いくつかの違いがあります：

- **引用符**：英語と同様に、ブラジルポルトガル語ではピリオドとコンマは
  引用符の内側に配置されます。
- **逆符号**：スペイン語とは異なり、ポルトガル語では逆疑問符（`¿`）や
  逆感嘆符（`¡`）を使用**しません**。
- **シリアルコンマ**：ポルトガル語ではリスト内の「e」の前にシリアルコンマを
  使用しません（例：「maçãs, laranjas e bananas」）。スペイン語と同様です。
- **数値形式**：英語では小数点としてピリオドを使用しますが（例：3.14）、
  ポルトガル語ではコンマを使用します（例：3,14）。千の位には、ポルトガル語では
  ピリオドを使用します（例：1.000）。
