Brazilian Portuguese (_português brasileiro_) is a Romance language descended
from [Vulgar Latin](https://en.wikipedia.org/wiki/Vulgar_Latin) and the official
language of Brazil, the largest country in both South America and the
[Lusophone](https://en.wikipedia.org/wiki/Lusophone) world. Portuguese and
Spanish share many similarities as closely related [Romance
languages](https://en.wikipedia.org/wiki/Romance_languages), so most of the
common foundations of YAMLResume can be reused for Brazilian Portuguese.

## Linguistics

### Character Set

Brazilian Portuguese uses the [Latin
alphabet](https://en.wikipedia.org/wiki/Latin_alphabet) with 26 letters (`a-z`).
Unlike Spanish, Portuguese does not have the letter `ñ`. Instead, the nasalized
equivalent is written as `nh`. Portuguese uses several diacritics for stress,
nasalization, and pronunciation:

| Feature                | English Character Set             | Brazilian Portuguese Character Set                                                                                          |
| ---------------------- | --------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| **Alphabet**           | 26 letters (`a-z`)                | 26 letters (`a-z`)                                                                                                          |
| **Diacritics**         | None                              | Includes acute (`á`, `é`, `í`, `ó`, `ú`), circumflex (`â`, `ê`, `ô`), grave (`à`), tilde (`ã`, `õ`), and diaeresis (`ü`)    |
| **Special Characters** | Limited to basic Latin characters | Includes `ç` (c-cedilha), which is not considered a separate letter but a variant of `c`                                     |

The main differences between the English and Brazilian Portuguese writing systems:

- **Diacritics**: Portuguese uses several diacritics that indicate stress and
  pronunciation changes. For example, acute accents (`é`) denote open vowels,
  circumflex accents (`ê`) denote close vowels, and tildes (`ã`, `õ`) denote
  nasalization.
- **Cedilla**: The `ç` (c-cedilha) is unique to Portuguese and appears only
  before `a`, `o`, and `u` (e.g., _caça_, _aço_, _açúcar_).
- **No inverted marks**: Unlike Spanish, Portuguese does not use inverted
  question marks (`¿`) or exclamation marks (`¡`).

## Idioms

### Date

Days of the week and months are not capitalised in Portuguese. For example, "Sep
2021" in English is translated as "set 2021" in Brazilian Portuguese. Month
names in abbreviated form are typically followed by a period (e.g., `jan.`).

### Capitalization

Portuguese follows conservative capitalization rules similar to Spanish. Months,
days of the week, languages, and nationalities are lowercase. Unlike German,
Portuguese does not capitalize all nouns.

YAMLResume internally adopts a fixed set of options and terms for all resume
templates. For example, languages in YAMLResume resume templates adopt
[Linkedin's language fluency
options](https://www.linkedin.com/pulse/language-proficiency-linkedin-duaa-ali/):

- Elementary Proficiency
- Limited Working Proficiency
- Minimum Professional Proficiency
- Full Professional Proficiency
- Native or Bilingual Proficiency

These are translated to Brazilian Portuguese as follows:

- Proficiência Básica
- Proficiência Profissional Limitada
- Proficiência Profissional Mínima
- Proficiência Profissional Fluente
- Nativo ou Bilíngue

Only the first word is capitalized.

## Typesetting

### Punctuations

Brazilian Portuguese shares several punctuation conventions with English, but
has some differences:

- **Quotation marks**: Similar to English, periods and commas are placed inside
  quotation marks in Brazilian Portuguese.
- **Inverted marks**: Unlike Spanish, Portuguese does **not** use inverted
  question marks (`¿`) or exclamation marks (`¡`).
- **Serial comma**: Portuguese does not use a serial comma before "e" in lists
  (e.g., "maçãs, laranjas e bananas"), similar to Spanish.
- **Numerical formats**: English uses a period as a decimal separator (e.g.,
  3.14), whereas Portuguese uses a comma (e.g., 3,14). For thousands, Portuguese
  uses a dot (e.g., 1.000).
