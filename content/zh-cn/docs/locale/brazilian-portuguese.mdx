title: 巴西葡萄牙语

巴西葡萄牙语（_português brasileiro_）是一种源自[通俗拉丁语](https://en.wikipedia.org/wiki/Vulgar_Latin)的罗曼语族语言，是南美洲最大国家、[葡萄牙语国家共同体](https://en.wikipedia.org/wiki/Lusophone)中最大国家巴西的官方语言。葡萄牙语和西班牙语作为密切相关的[罗曼语族语言](https://en.wikipedia.org/wiki/Romance_languages)有许多相似之处，因此 YAMLResume 的大部分通用基础可以复用于巴西葡萄牙语。

## 语言学

### 字符集

巴西葡萄牙语使用[拉丁字母](https://en.wikipedia.org/wiki/Latin_alphabet)，共有 26 个字母（`a-z`）。
与西班牙语不同，葡萄牙语没有字母 `ñ`。取而代之的是，鼻音化的等价形式写作
`nh`。葡萄牙语使用多种变音符号来表示重音、鼻音化和发音：

| Feature                | English Character Set             | Brazilian Portuguese Character Set                                                                                          |
| ---------------------- | --------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| **Alphabet**           | 26 letters (`a-z`)                | 26 letters (`a-z`)                                                                                                          |
| **Diacritics**         | None                              | Includes acute (`á`, `é`, `í`, `ó`, `ú`), circumflex (`â`, `ê`, `ô`), grave (`à`), tilde (`ã`, `õ`), and diaeresis (`ü`)    |
| **Special Characters** | Limited to basic Latin characters | Includes `ç` (c-cedilha), which is not considered a separate letter but a variant of `c`                                     |

英语和巴西葡萄牙语书写系统的主要区别：

- **变音符号**：葡萄牙语使用多种变音符号来表示重音和发音变化。例如，尖音符（`é`）
  表示开元音，抑扬符（`ê`）表示闭元音，波浪号（`ã`、`õ`）表示鼻音化。
- **软音符**：`ç`（c-cedilha）是葡萄牙语特有的，只出现在 `a`、`o` 和 `u` 之前
  （例如 _caça_、_aço_、_açúcar_）。
- **无倒置标点**：与西班牙语不同，葡萄牙语不使用倒问号（`¿`）或倒感叹号（`¡`）。

## 习语

### 日期

葡萄牙语中星期和月份不大写。例如，英语中的 "Sep 2021" 在巴西葡萄牙语中翻译为
"set 2021"。缩写形式的月份名称通常后跟句点（例如 `jan.`）。

### 大写

葡萄牙语遵循与西班牙语类似的保守大写规则。月份、星期、语言和国籍均为小写。
与德语不同，葡萄牙语不将所有名词大写。

YAMLResume 内部为所有简历模板采用固定的选项和术语集。例如，YAMLResume 简历模板中的
语言采用 [LinkedIn 语言熟练程度

只有第一个单词大写。

## 排版

### 标点符号

巴西葡萄牙语与英语共享多种标点符号规范，但也有一些不同之处：

- **引号**：与英语类似，巴西葡萄牙语中句号和逗号放在引号内。
- **倒置符号**：与西班牙语不同，葡萄牙语**不**使用倒问号（`¿`）或倒感叹号（`¡`）。
- **序列逗号**：葡萄牙语在列表中的 "e" 之前不使用序列逗号
  （例如 "maçãs, laranjas e bananas"），与西班牙语类似。
- **数字格式**：英语使用句点作为小数点分隔符（例如 3.14），而葡萄牙语使用逗号
  （例如 3,14）。千分位则使用句点（例如 1.000）。
