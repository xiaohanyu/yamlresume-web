---
title: 佈局
---

履歷的核心是內容，但佈局也很重要。許多基於 JSON/YAML 的履歷生成器只關注內容，而將佈局保持不變，這給用戶留下了很大的問題。

與其他履歷生成器不同，YAMLResume 提供了開箱即用的佈局解決方案，允許用戶專注於履歷的內容，同時生成具有像素完美佈局和排版的專業履歷。

以下是 `resume.yml` 文件中佈局配置的示例：

```yml lineNumbers title="resume.yml"
---
content:
  # ...

layout:
  locale:
    language: en
  margins:
    top: 2.5cm
    left: 1.5cm
    right: 1.5cm
    bottom: 2.5cm
  page:
    showPageNumbers: true
  template: moderncv-banking
  typography:
    fontSize: 11pt
```

## 語言環境

`locale` 選項允許您為履歷指定語言。這將影響日期格式和其他特定於語言環境的設置。

```yml lineNumbers
layout:
  locale:
    language: en
```

查看[多語言](/zh-tw/docs/content/multi-languages)了解更多詳情。

## 邊距

`margins` 選項允許您自定義頁面邊距。

```yml lineNumbers
layout:
  margins:
    top: 2.5cm
    left: 1.5cm
    right: 1.5cm
    bottom: 2.5cm
```

## 頁面

`page` 選項允許您自定義頁面設置，如紙張大小和是否顯示頁碼。

```yml lineNumbers
layout:
  page:
    showPageNumbers: true
```

## 章節

`sections` 選項允許您自定義章節標題及其順序。更多資訊，請參見[章節文件](./layout/sections)。

```yml lineNumbers
layout:
  sections:
    aliases:
      work: "Professional Experience"
      education: "Education"
      skills: "Skills"
    order:
      - basics
      - work
      - education
```


## 模板

`template` 選項允許您從各種模板中選擇。每個模板都有獨特的設計和佈局。您可以在[模板文件](./layout/templates)中找到可用模板的列表。

```yml lineNumbers
layout:
  template: moderncv-banking
```

## 排版

`typography` 選項允許您自定義履歷中使用的字體樣式。

目前，只支援 `fontSize`，LaTeX 引擎只支援 `10pt`、`11pt` 和 `12pt`。

```yml lineNumbers
layout:
  typography:
    fontSize: 11pt
```
